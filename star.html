태그의 종류

strong-글씨를 진하게
u-밑줄
h-각 세션의 제목 h1~h6까지 존재
br-줄바꿈
단 br은 닫힌 태그가 존재하지않음(ex)br.hr,input,img 등)
p-단락
단락과 단락과의 간격이 고정되어있음
css를 이용해 간격수정가능
ex)p style="margin-top:45px;":간격을 45px만큼 조정.
ol-숫자로된목차(자동정렬가능)
ul-목록
li-목차
body-본문
head-본문을 설명하는 태그
title-제목지정
a-링크지정

태그의 속성

태그a의 속성

  1. href="링크 주소"
  2. target="_blank" 새탭으로 열기
  3. title=""링크가 어떤 내용을 담고 있는지를 툴팁으로 보여주는 기능.

태그 간의 관계

부모(parent),자식(child)필요에 따라서 태그 간 부모,자식이 달라짐
단, 고정된 관계가 존재

  • ul-li
  • ol-li

태그의 구조

  • title 검색엔진이 웹페이지를 분석시 가장 중요하게 생각하는 태그
    -영어가 아닌 문자로 웹페이지를 만들시 에디터에 있는 엔코딩 참고
  • meta charset="utf-8" 필수로 입력
  • body 본문
  • head 본문을 설명하는 태그
  • html body태그와 head태그를 감싸주는 태그
  • !doctype html HTML로서 만들어졌다는 것을 표현하기 위해서 문서의 시작에 추가

웹의 역사와 웹의 구동 원리

웹 브라우저는 웹 서버에 정보를 요청하고, 웹 서버는 그 요청한 정보를 하드디스크에서 검색해서 다시 웹 브라우저에 응답한다.
